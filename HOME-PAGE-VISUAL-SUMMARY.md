## Responsive Breakpoints

### Mobile (< 640px)

- All grids stack to 1 column
- Hero: Text above, image below
- Services: Single column cards
- Process: Vertical timeline (no connector line)
- Why Choose Us: Features above, BOQ + stats below
- Stats: Remains 2x2 grid
- Cities: Wrap naturally
- FAQ: Full width accordion

### Tablet (640px - 1024px)

- Hero: Side-by-side text + image
- Services: 2-column grid
- Finishing: 3-column grid
- Process: 2-column grid
- Why Choose Us: Starts to side-by-side
- Cities: Multiple per row

### Desktop (> 1024px)

- Hero: Full 2-column with spacing
- Services: 3-column grid
- Finishing: 6-column grid
- Process: 5-column with connector line
- Why Choose Us: Full 2-column layout
- All spacing optimized (lg:py-24)
